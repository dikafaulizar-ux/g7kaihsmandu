## 🚀 Cara Deploy ke Vercel

### 1. Upload ke GitHub
1. Buka github.com → buat repository baru: `g7kaih-sman2`
2. Upload SEMUA file dan folder ini ke repository

### 2. Deploy ke Vercel
1. Buka vercel.com → Sign up dengan GitHub
2. Klik "Add New Project"
3. Pilih repository `g7kaih-sman2`
4. Klik "Deploy"
5. Selesai! Dapat link publik ✅
